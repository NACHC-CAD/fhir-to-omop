Race and Terminology Mapping in the NACHC fhir-to-omop Tools

Summary

The FHIR US Core Profile uses some internally defined terminologies including a terminology for Race/Ethnicity. The terminology can be found here . The US Core Profile Implementation guide is here . Section 10.0.7 Contains information about the terminologies used by U.S. Core including Race and Ethnicity. The Race and Ethnicity code systems were downloaded as XML from here as XML and the XML was converted to CSV files. These CSV files were then mapped to existing OMOP codes. The files used in this process are shown below.

Data Files

FHIR: Race & Ethnicity This is the source file for all of the Race and Ethnicity codes from 10.143.1 CodeSystem: Race & Ethnicity - CDC. This file was then divided into a race file and a ethnicity file.
FHIR: Race This is the file that contains just the Race data from 10.143.1 CodeSystem: Race & Ethnicity - CDC.
FHIR: Ethnicity This is the file that contains just the Ethnicity data from 10.143.1 CodeSystem: Race & Ethnicity - CDC.
OMOP: Race This is an extract of the Race concepts from the OMOP Concept table (data down loaded from Athena).
OMOP: Ethnicity This is an extract of the Race concepts from the OMOP Concept table (data down loaded from Athena).

Mapping Files

The following mapping files were manually mapped using the above data files.
Mapping: Race This file contains the mappings of the Race codes used by FHIR to the race codes used by OMOP.
Mapping: Ethnicity This file contains the mappings of the Ethnicity codes used by FHIR to the race codes used by OMOP.
