Developer How Tos

This page provides a list of resources for developers wishing to use or contribute to the fhir-to-omop project (and some other generally useful stuff).

Resource Description
Git

Getting Started with Git and Github
This is the quick start guid for Git and Git hub. Git is a version control system that is used by virtually all open source (and other large) projects. This quick start guide shows how to download and install a Git client, how to create a Github account, and how to get permissions for an existing Github project (like this one!).

Git Compare branch/fork
How to compare two different branches and or forks using the github web interface. Spoiler alert: you need to pretend you're creating a pull request.
Git Add HTML Documents
How to add html based documents to a github repository.
IDE (Eclipse) Install a Fully Automated, Fully Featured Eclipse Environment
This how-to describes how to install a fully configured and fully featured Eclipse environment. This environment can be installed by simply downloading a single zip file and unzipping it on your local machine. This install is for Windows.

This Install includes the following
  • Eclipse
  • Git and Git Integration
  • Maven and Maven Integration
  • Wildfly and Wildfly Integration
  • Lombok and Lobmok Integration
IDE (Eclipse) Install, Run, and Debug a Web Application in Eclipse
This how-to will show you how to download source code for a web application and run and debug that web application locally.
