Create a Wildfly Datasource (MySql)

This document details how to deploy and configure a datasource (database connection) for MySql.
To create a datasource simply add something similar to the following to standalone.xml. Note that this datasource uses the jar file module created using the instructions here.

You will need to modify the URL and password to use the valid values for your system. 

<datasource jndi-name="java:/MySqlDS" pool-name="MySqlDS">
    <connection-url>jdbc:mysql://localhost:3306/cosmos?rewriteBatchedStatements=true&useJDBCCompliantTimezoneShift=true&useLegacyDatetimeCode=false&serverTimezone=UTC</connection-url>
    <driver>mysql</driver>
    <security>
        <user-name>cosmos</user-name>
        <password>MySecretPassword</password>
    </security>
    <validation>
        <valid-connection-checker class-name="org.jboss.jca.adapters.jdbc.extensions.mysql.MySQLValidConnectionChecker"/>
        <background-validation>true</background-validation>
        <exception-sorter class-name="org.jboss.jca.adapters.jdbc.extensions.mysql.MySQLExceptionSorter"/>
    </validation>
</datasource>
Navigate to Configuration->Subsystems->Datasources & Drivers->Datasources. You should now see the MySqlDS.



Select View->Test Connection to test the connection.
