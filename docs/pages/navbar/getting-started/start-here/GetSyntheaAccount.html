Get a Synthea (Synthmass) Account and Application ID

To connect to Synthea/Synthmass we will need a Synthea account and an application Id.

Get a Synthea Account

Instructions on how to get a Synthea account and application key are at https://synthea.mitre.org/api-keys



Get an Application ID

To get an application id, select the APPS selection in the upper right corner under your account name.

Select "+NEW APP" in the upper right corner, fill out the form, and submit.

Your response from Synthea should look something like what is shown below.

Be sure to record your App ID, API Key, and Secret.
