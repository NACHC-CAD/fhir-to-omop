Install R and RStudio

R and RStudio are used by the OHDSI Data Quality Dashboard and other OHDSI tools. The instructions here will guide you through the installation of R and RStudio.

Install R

R can be downloaded and installed from https://cran.r-project.org/bin/windows/base/

Install R Tools

R Tools can be downloaded and installed from https://cran.r-project.org/bin/windows/Rtools/rtools40.html

Install RStudio

R can be downloaded and installed from https://www.rstudio.com/products/rstudio/download/

The first time you launch R you will be prompted if you want to install devtools. Select Yes.
