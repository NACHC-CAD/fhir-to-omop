Create A UMLS Account

Some of the terminologies used by OMOP require a UMLS account and API key. To get a UMLS account and API key follow the steps below.

Before you can create a UMLS account you will need an identity provider. I'm using login.gov. To create a login.gov account follow the instructions here.

Create Your Account

Submit a request for a UMLS account from https://uts.nlm.nih.gov/uts/signup-login/ and select "Sign Up".

Once your done filling out the forms you will be told it will take 3 business days for your request to be processed
