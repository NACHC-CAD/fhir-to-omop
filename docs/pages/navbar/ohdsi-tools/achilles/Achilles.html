Install, Run, and View Achilles

Achilles is an integral component of Atlas (Data on the Data Source tab of Atlas are generated from Achilles for the most part). You probably want to install Atlas. If that is the case, skip this page and go directly to the Atlas Install Instructions page.

Achilles is a data quality tool. The github page for Achilles is here. This document is based on the documentation available for Achilles here. Achilles can be completely installed an run on windows following the instructions on this page.

To install and run Achilles run the following scripts.

  • install-achilles.r: Run the R script here to install components required for Achilles. This script only needs to be run once.
  • init-params-achilles.r: Run the R script here to initialise the parameters for Achilles.
  • run-achilles.r: Run the R script here to run Achilles.
  • View Achilles Data: Achilles data have been integrated into Atlas. To install Atlas, follow the instructions here: Install Atlas.
