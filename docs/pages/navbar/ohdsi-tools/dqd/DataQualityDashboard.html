Install The OHDHSI Data Quality Dashboard (DQD)


The instructions here will guide you through the installation of the OHDSI Data Quality Dashboard (DQD).
The Data Quality Dashboard is a tool written in R that can perform a number of configurable tests on your OMOP instance. More information is available at
The DQD collects meta data for your instance by running an R script that generates a single JSON file. This file is then read as a web page using any browser.
The instructions here show how to view your DQD information using a server that is launched from R.
